# 🎓 Scholarship Eligibility System (Python)

# 📌 Description
A Python-based beginner-friendly mini project that evaluates a student's scholarship eligibility based on multiple conditions like marks, income, category, extracurricular activities, and backlogs.
It demonstrates how real-world decision systems can be built using logical conditions.

# ⚙️ Features
- Checks eligibility using multiple inputs
- Categorizes students into:
  - 🟢 Full Scholarship (100%)
  - 🔵 Special Category Scholarship (70%)
  - 🟡 Partial Scholarship (50%)
  - ❌ Rejected / Not Eligible
- Uses nested if-else with AND / OR logic

# 🧠 Concepts Used
- Conditional Statements (`if-elif-else`)
- Logical Operators (`and`, `or`)
- List membership (`in`)
- User Input Handling

# ▶️ How to Run
1. Open the Python file
2. Run the program
3. Enter required inputs:
   - Marks
   - Income
   - Category
   - Extra-curricular (yes/no)
   - Backlogs (yes/no)
4. View the result

# 💡 Learning Outcome
- Importance of condition priority
- Difference between AND vs OR logic
- Writing clean and readable conditions

# 💻 Tech Used
Python (Beginner Level)
